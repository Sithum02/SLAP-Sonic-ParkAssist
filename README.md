# SLAPSonic ParkAssist

SLAPSonic ParkAssist is an aftermarket parking assist solution designed for vehicles that lack built-in parking sensors. The product provides distance and direction indicators using ultrasonic sensors and buzzers, offering a more intuitive and effective parking experience.

## Project Overview

The SLAPSonic ParkAssist system uses the ATmega328P microcontroller integrated with JSN-SR04T waterproof ultrasonic sensors to measure distances to obstacles and activate buzzers to alert the driver. The system is designed to be easy to install, cost-effective, and reliable for vehicles that require parking assistance.

### Key Features:
- **Microcontroller:** ATmega328P, providing the necessary computing power to handle multiple sensors and buzzers simultaneously.
- **Sensors:** JSN-SR04T waterproof ultrasonic sensors with a wide measuring range and angle.
- **Buzzers:** Active buzzers for audible alerts, with plans to introduce directional tones in future iterations.
- **Enclosure:** Custom-designed enclosure 3D printed in PLA filament, ensuring durability and sleek aesthetics.

### System Components:
- **Power Supply:** LM7805 voltage regulator for stable 12V to 5V conversion.
- **Main PCB:** The PCB connects all modules and integrates the microcontroller.
- **Sensors:** 8 JSN-SR04T sensors to measure distances from obstacles.
- **Buzzers:** Active buzzers for obstacle alerts.

## Features

- **Distance and Direction Alerts:** Provides both distance and direction information using multiple buzzers for clear guidance.
- **Waterproof Sensors:** JSN-SR04T sensors are waterproof, making them suitable for outdoor vehicle installations.
- **Compact and Reliable Design:** The product architecture is designed to reduce size and improve cost-effectiveness for wider adoption.

## Future Improvements

- **PCB Integration:** We aim to integrate the sensor modules directly into the main PCB to reduce the enclosure size and production costs.
- **Injection Molding Enclosure:** Future iterations of the enclosure will be produced using polycarbonate for improved impact and thermal resistance.
- **Microcontroller Upgrade:** Transition to STM microcontrollers for better performance and power management.

## Installation Guide

The SLAPSonic ParkAssist system is designed for easy installation:
1. Mount the ultrasonic sensors at appropriate locations on your vehicle.
2. Connect the sensors and buzzers to the main PCB.
3. Use the provided instructions to install the system and connect it to the vehicle's power supply.
4. Test the system to ensure proper functionality.
